DESCRIPTION:

Wrapper around ruby http and https libraries.

Converted to a gem from a mixture of versions I’ve used over the years. Might need some work yet :)

FEATURES/PROBLEMS:

  • session block
  • get, post, put, delete

SYNOPSIS:

call with WebBrowser.http_session() or https_session(). e.g. WebBrowser.https_session( host: @hostname, verify_cert: false ) do |ws| result = ws.get_page( query: "#{api_query}", authorization: "token #{@auth_token}", form_values: { "page_size"=>page_size, "page"=>page, }.merge(args) ) end

If the server supports keeping the connection open, then the block can have multiple calls per session.

REQUIREMENTS:

INSTALL:

  • sudo gem install wikk_webbrowser
